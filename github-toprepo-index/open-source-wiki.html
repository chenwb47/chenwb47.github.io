GitDiagram

代码可视化

AI 驱动的 GitHub 仓库可视化工具，可将任何代码库转换为交互式架构图。只需将 GitHub URL 中的 "hub" 替换为 "diagram" 即可 instantly 生成图表。

将仓库结构转为交互式系统架构图
点击组件直接跳转到对应源代码
基于 OpenAI o4-mini 快速生成
支持自定义指令重新生成图表
⭐ 2.7k 🔧 TypeScript
官网 GitHub
🧜

Mermaid

图表绘制

基于 JavaScript 的图表绘制工具，使用 Markdown 风格的文本定义动态创建和修改图表。支持流程图、时序图、甘特图等多种图表类型。

支持流程图、时序图、类图、ER图等
GitHub、GitLab、Notion 原生支持
Markdown 集成，版本控制友好
2019 JS Open Source Award 获奖项目
⭐ 85k+ 🔧 JavaScript
文档 GitHub

一键运行开源大语言模型
支持 Llama 3、Mistral、CodeLlama 等
支持自定义和创建模型
REST API 和命令行工具
⭐ 52k+ 🔧 Go
官网 GitHub

模块化组件构建 LLM 应用
支持多种模型和向量数据库
内置监控和评估工具
支持转换为 REST API
⭐ 96k+ 🔧 Python/JS

Mermaid 示例 Ollama 安装
🍎 macOS
# 使用 Homebrew 安装
brew install ollama

# 或者下载安装包
# https://ollama.com/download
🐧 Linux
# 一键安装脚本
curl -fsSL https://ollama.com/install.sh | sh

# 启动服务
sudo systemctl start ollama
🪟 Windows
# 下载 Windows 安装程序
# https://ollama.com/download/windows

# 或使用 winget
winget install Ollama.Ollama
🚀 快速使用
# 拉取并运行 Llama 3
ollama run llama3

# 列出本地模型
ollama list

# 启动 REST API 服务 (默认端口 11434)
ollama serve
